# 红叶网络验证环境

部署本验证系统需要用户自备服务器(非虚拟主机)，Windows或Linux均可，推荐使用Linux-Centos7服务器

## 最低服务器配置要求：  
服务器具体需要什么配置，取决于您的用户规模以及心跳策略，以下数据仅供参考，请自行测试
::: tip
同时在线用户数100+：2C4G2M(最低) 2C8G10M(建议)  
同时在线用户数1000+：4C8G10M(最低) 4C16G20M(建议)  
同时在线用户数5000+：8C16G20M(最低) 8C32G100M(建议)  
同时在线用户数10000+：8C32G50M(最低) 8C64G100M(建议)  
:::

## 服务器需安装以下运行环境
::: tip
1. JDK >= 1.8 (建议1.8版本)
2. Mysql >= 5.7.0 (建议5.7版本)
3. Redis >= 5.0 (建议7.0版本)
4. Nginx >= 1.20.0
:::
