# 激活授权

:::tip 关于授权机制  
1. 授权基于本地授权，授权状态不会因官方服务器宕机而失效，请放心使用。  
2. 系统授权基于服务器硬件信息，如：CPU、内存、硬盘、网卡、BIOS、MAC等，授权状态会因硬件信息变更而失效，请及时更新授权。  
3. 同一授权码不同服务器仅可激活一次，如需激活不同服务器，请重新购买授权。  
4. 授权失效后，系统将限制管理后台登录，软件用户数量限制为100人/软件，重新续费激活后限制解除。  
:::

:::tip 授权有效期内  
1. 同一授权码同一服务器可无限次激活，若因故丢失了授权，请于官网重新激活即可。  
2. 使用新授权码再次激活授权，系统将自动将授权时长累加，无须担心授权时长覆盖。  
3. 系统可随时更新，无需重新授权。  
4. 软件、卡密、会员、代理等所有功能均无数量限制及额外收费。  
5. 授权有效期内可随意更换访问端口及域名，无需重新授权。  
:::
